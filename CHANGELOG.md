# Changelog

LookMD 的重要版本变更记录在此文件中。

本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) 进行版本管理。

## [0.2.0] - 2026-07-14

### 新增

- 新增所见即所得 Markdown 编辑模式，并默认启用。
- 新增编辑格式工具栏，支持标题、粗体、斜体、删除线、代码块、列表、引用和链接。
- 新增 Markdown 快捷输入：支持标题、列表、引用、代码块、粗体、斜体、删除线、行内代码和链接。
- 支持在块级快捷格式转换后立即按 Backspace 恢复原始 Markdown 前缀。
- 在设置中新增“所见即所得 / Markdown 源码”默认编辑模式选项。

### 修复

- 修复无序列表与有序列表显示样式相同的问题。
- 修复切换文件时阅读与编辑滚动位置可能互相影响的问题。
- 修复搜索框状态同步和中文输入法组合输入问题。
- 修复编辑历史状态在渲染期间读取 Ref 的问题。

### 改进

- 所见即所得内容保存时自动转换为标准 Markdown，并保留常用 GFM 格式与相对图片路径。
- ESLint 排除 Tauri 和前端构建产物，项目级代码检查可正常运行。
- 统一前端、Tauri 配置与 Rust crate 的版本号。
